## RNCについて

RNC（Rehabilitation Nutrition Community）は、

* リハビリテーション栄養
* フレイル・サルコペニア
* ポリファーマシー
* 高齢者薬物療法
* せん妄
* 医療安全

に関する学習・教育・実践支援を目的として活動しています。

本リポジトリでは、医療従事者向けに作成した各種コンテンツを公開しています。
