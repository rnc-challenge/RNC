# RNC
# RNC Open Resources

リハ栄養コミュニティ（RNC）が公開する教育資材・臨床支援ツール・Visual Abstract集です。

---

## RNCについて

RNC（Rehabilitation Nutrition Community）は、

* リハビリテーション栄養
* フレイル・サルコペニア
* ポリファーマシー
* 高齢者薬物療法
* せん妄
* 医療安全

に関する学習・教育・実践支援を目的として活動しています。

本リポジトリでは、医療従事者向けに作成した各種コンテンツを公開しています。

## Clinical Tools

臨床現場で利用できる支援ツールです。

### 薬効群別 リハ・生活機能確認支援ツール

* SOAP作成
* 多職種申し送り
* リスク確認

を支援します。
